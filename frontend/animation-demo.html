Public API

Use these methods to trigger animations programmatically:

MicroInteractions.showSuccess(element) - Show success checkmark

MicroInteractions.showError(element) - Show error shake

MicroInteractions.addShimmer(element) - Add shimmer effect

MicroInteractions.addPulse(element) - Add pulse ring

MicroInteractions.createRipple(event) - Create ripple at click point
